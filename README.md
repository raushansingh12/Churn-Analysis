# Churn-Analysis
End-to-end Customer Churn Analysis using SQL, SQLite, Python, Pandas, NumPy, Matplotlib, and Seaborn. Analyzed 10,000+ customer records to perform data cleaning, EDA, feature engineering, KPI analysis, customer segmentation, churn trend analysis, and business insight generation through interactive visualizations.

**🎯 Business Problem**
Customer churn directly impacts company revenue and profitability.
This project helps answer questions such as:

1. Which customers are more likely to churn?
2. Which subscription plans have the highest churn?
3. Which states contribute the highest revenue?
4. How do customer complaints affect churn?
5. What trends exist in customer cancellations over time?

**📊 Dataset**

The project consists of three relational SQLite tables:

**Customer Table**
Customer ID,
Name,
Gender,
State,
Country,
DOB,
Interests.

**Subscription Table**
Subscription Type,
Plan Type,
Contract Type,
Monthly Charges,
CLTV,
Churn Score,
Cancellation Date.

**Support Table**
Complaint Date,
Escalations,
CSAT Score,
Customer Comments.

**Dataset Size**
10,000+ Customers,
3 SQL Tables.

**Technologies Used**
Python,
SQL,
Pandas,
NumPy,
Matplotlib,
Seaborn.
Jupyter Notebook.

**📈 Project Workflow**
SQLite Database -> SQL Queries -> Pandas  -> Data Cleaning  -> Data Cleaning -> Feature Engineering -> Exploratory Data Analysis -> Business KPI Analysis -> Visualization -> Business Insights

**💡 Business Insights**
* Identified customer segments with the highest churn probability.
* Compared churn across subscription and contract types.
* Analyzed the relationship between customer satisfaction and churn.
* Evaluated customer lifetime value across different customer segments.
* Generated actionable insights to support customer retention strategies.
