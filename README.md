# Churn-Analysis
End-to-end Customer Churn Analysis using SQL, SQLite, Python, Pandas, NumPy, Matplotlib, and Seaborn. Analyzed 10,000+ customer records to perform data cleaning, EDA, feature engineering, KPI analysis, customer segmentation, churn trend analysis, and business insight generation through interactive visualizations.

**🎯 Business Problem**
Customer churn directly impacts company revenue and profitability.
This project helps answer questions such as:

1. Which customers are more likely to churn?
2. Which subscription plans have the highest churn?
3. Which states contribute the highest revenue?
4. How do customer complaints affect churn?
5. What trends exist in customer cancellations over time?
